## Installation

### Prerequisites

- **Node.js** (v14 or above)
- **npm** or **yarn**

### Steps to Set Up

1. **Clone the repository**:
    ```bash
    git clone https://github.com/your-username/Recipe-Website.git
    ```

2. **Navigate to the project directory**:
    ```bash
    cd Recipe-Website
    ```

3. **Install the dependencies**:
    ```bash
    npm install
    ```
    or
    ```bash
    yarn install
    ```

4. **Run the development server**:
    ```bash
    npm start
    ```
    or
    ```bash
    yarn start
    ```
